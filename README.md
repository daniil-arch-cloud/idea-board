# Idea Board

Idea Board — локальное desktop-приложение для визуальной организации идей, заметок, рисунков, изображений и связей между объектами.

Проект представляет собой интерактивную доску, где можно создавать объекты, соединять их разными типами линий, группировать, перемещать по рабочей области и сохранять данные локально на компьютере.

## Возможности

- создание и редактирование заметок;
- создание листов для рисования;
- вставка изображений из буфера обмена;
- создание нескольких независимых досок;
- соединение объектов линиями и стрелками;
- поддержка прямых, пунктирных, тонких, толстых и кривых связей;
- управление изгибом кривой линии через отдельную точку;
- ручная настройка начала и конца связи;
- узлы связи для создания разветвлённых схем;
- группировка и разгруппировка объектов;
- перемещение и масштабирование рабочей области;
- горячие клавиши для быстрых действий;
- несколько тем оформления;
- локальное сохранение данных;
- экспорт и импорт данных проекта;
- сборка desktop-приложения под Windows через Electron.

Руководство по доске
Краткая инструкция по основным функциям редактора

Основные объекты

На доске можно создавать заметки, листы для рисования, картинки и узлы связи. Все объекты можно перемещать мышкой, выделять и соединять линиями.

Nсоздать заметку
Lсоздать лист для рисования
Ctrl + Vвставить картинку
Hоткрыть или закрыть руководство

Заметки

У заметки можно менять заголовок, текст, цвет и размер. Заметки можно двигать по доске, связывать с другими объектами и добавлять в группы.

  • Клик по заметке — выбрать заметку.
  • Перетаскивание за верхнюю часть — переместить заметку.
  • Правый нижний угол — изменить размер.
  • Кнопка удаления удаляет заметку вместе с её связями.

Листы для рисования

Лист для рисования — это белая область, на которой можно делать наброски. Его можно двигать, соединять с другими объектами и использовать как визуальную заметку.

1кисть
2ластик
Rочистить активный лист

Связи и линии

В режиме связи можно соединять любые объекты: заметки, картинки, листы и узлы. Для создания связи включи режим связи и выбери два объекта.

  • Линия может быть обычной, со стрелкой, двусторонней, пунктирной, тонкой или толстой.
  • Кривую линию можно изгибать за центральную точку.
  • Начало и конец связи можно двигать по краю объекта.
  • Правая кнопка по ручке начала или конца возвращает автоматическую привязку.
  • Правая кнопка по ручке кривой возвращает автоматический изгиб.
Cвключить режим связи
Escвыйти из режима связи

Узлы связи

Узел связи — это маленькая точка-разветвитель. К ней можно подключать несколько линий, чтобы делать схемы вида: один объект → узел → несколько объектов.

  • Все линии, которые входят в узел или выходят из него, крепятся к центру узла.
  • Узел можно перемещать как обычный объект.
  • Узел можно соединять с заметками, картинками, листами и другими узлами.
  • Выбранный узел можно удалить клавишей Delete или Backspace.
  • Правый клик по узлу удаляет его сразу.

Группы

Несколько объектов можно объединять в группу, чтобы перемещать их вместе.

Shift + кликвыбрать объект для группы
Gсоздать группу
Uразгруппировать

Навигация и доски

Можно создавать несколько досок, переключаться между ними, экспортировать данные и импортировать их обратно. В списке объектов отображаются закреплённые элементы, а через поиск можно найти любой объект по названию.

Bоткрыть/закрыть левую панель
Dоткрыть/закрыть панель досок
Колесо мышимасштаб доски
Перетаскивание пустого местадвигать камеру

История действий

Основные действия можно отменять и возвращать.

Ctrl + Zотменить действие
Ctrl + Xвернуть действие
Ctrl + Cкопировать объект
Ctrl + Vвставить объект или картинку

Интерфейс

Сверху находится мини-панель статуса: она показывает название текущей доски, количество объектов, количество связей и масштаб. Там же можно быстро приблизить, отдалить или центрировать доску.

  • Пустая доска показывает стартовую подсказку.
  • Уведомления внизу показывают создание, удаление и режимы работы.
  • После удаления объекта в уведомлении появляется кнопка × — она отменяет удаление.
  • Тема оформления меняется в левой панели.
